# GoReact Chatroom
A real time chatroom application built with Go and React

## Requirements
- [Golang](https://golang.org/dl)
- [Gorilla/Websocket](https://github.com/gorilla/websocket)
- [React](https://reactjs.org)
- [NodeJS](https://nodejs.org/en/)

## Usage

- [Golang Server](https://github.com/jjohnston55/GoReact-Chat/tree/master/backend)
- [React Client](https://github.com/jjohnston55/GoReact-Chat/tree/master/frontend)
